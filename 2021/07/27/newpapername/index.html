## leecode1hashmap最优
通过把数组中的值当做map中的key，下标（i）当做map中的value，用target减数组中的值去得到一个diff，然后判断diff在map中是否有对应的key和其大小一样，有则返回，没有则添加到hashmap中。

    public class LeeCode1HashMap {
    public static int[] twosum(int[] nums,int target){
        HashMap<Integer,Integer> map=new HashMap<>();
        for (int i=0;i<nums.length;i++){
            int diff=target-nums[i];
            if (map.containsKey(diff)){
                return new int[] {map.get(diff),i};
            }else {
                map.put(nums[i],i);
            }
        }
        return new int[] {-1,-1};
    }
    public static void main(String[] args) {
        int[] nums={3,2,11,7};
        int target=9;
        System.out.println(Arrays.deepToString(new int[][]{(twosum(nums, target))}));
    }
}
